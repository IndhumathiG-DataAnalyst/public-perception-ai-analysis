# Public Perception & Trend Analysis of Artificial Intelligence

## Tools Used
- Python
- Power BI

## Project Description
This project analyzes public perception and emerging trends related to Artificial Intelligence.
Public opinion data was cleaned and analyzed using Python, and key insights were visualized
through interactive Power BI dashboards.

## Objectives
- Understand public sentiment towards AI
- Identify key trends and patterns in AI adoption
- Visualize insights for better interpretation

## Key Insights
- Identified positive, negative, and neutral perceptions of AI
- Observed trending topics related to AI usage and impact
- Created dashboards to track perception trends over time
