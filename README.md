# Public Perception & Trend Analysis of Artificial Intelligence

## Description
Python and Power BI based trend analysis on AI public perception.  
This project analyzes public opinion data related to Artificial Intelligence,  
cleans and processes it using Python, and visualizes key insights with Power BI.

## Tools Used
- Python (Pandas, NumPy, Matplotlib/Seaborn)
- Power BI

## Project Highlights
- Collected and cleaned public perception data related to AI.
- Performed sentiment analysis and trend detection using Python.
- Built interactive Power BI dashboards to visualize insights clearly.
- Identified key positive and negative themes related to AI public opinion.
